Caddie Master

Gestione attrezzature deposito golf

Cos'e' Caddie Master

Caddie Master e' un'app mobile standalone per gestire la preparazione delle attrezzature da golf (sacche su carrelli) prima di una gara. Il deposito e' organizzato per file numerate, e ogni giocatore ha la propria attrezzatura in una fila specifica.

L'app permette al caddie master di sapere, data una lista di partenze, quali file aprire e quali attrezzature preparare, ottimizzando i viaggi al deposito.

Offline Dati sul dispositivo Nessun server

Le 4 sezioni

L'app ha una barra di navigazione in basso con 4 tab:

+ Importa ⏰ Gara ☰ File 👤 Anagrafica

1. Importa Gara

Qui si importa la lista delle partenze di una gara.

Come funziona

1
Inserisci i dati
Compila nome gara, data, e incolla il testo della lista partenze nella textarea.
2
Premi "Analizza"
L'app parsa il testo e confronta i nomi con l'anagrafica. Ogni giocatore viene segnato come: Noto (fila gia' assegnata) oppure Nuovo (non in anagrafica).
3
Assegna le file ai nuovi (opzionale)
Per i giocatori nuovi puoi inserire il numero della fila. Se non lo conosci, puoi lasciarlo vuoto: il giocatore verra' omesso dalla preparazione attrezzature.
4
Premi "Conferma e Salva"
La gara viene salvata. I giocatori nuovi a cui hai assegnato una fila vengono aggiunti automaticamente all'anagrafica.

Formato lista partenze

Il testo deve avere questo formato (campi separati da TAB):

08:36  1  COGNOME NOME 15  COGNOME NOME 17
08:45  1  COGNOME NOME 12  COGNOME NOME 8  COGNOME NOME 17

Dove: ORARIO TEE GIOCATORE1 GIOCATORE2 ...

Il numero dopo il nome e' l'handicap (viene ignorato). Le voci "Privacy" vengono saltate.

Gare precedenti

Le gare gia' importate appaiono in cima alla pagina. Puoi selezionarle per rivederle oppure eliminarle con il bottone Elimina.

2. Gara

Mostra le partenze della gara selezionata, raggruppate per orario.

Funzionalita'

  • Ogni gruppo mostra orario e numero del tee
  • Accanto a ogni giocatore c'e' il N. fila
  • I giocatori senza fila mostrano No fila
  • Tap su un giocatore: evidenzia in verde tutti i giocatori della stessa fila, in tutte le partenze
  • Tap di nuovo per deselezionare

Questa vista e' utile per capire rapidamente "chi parte con chi" e in quale fila si trovano.

3. File Deposito

Questa e' la vista principale per il lavoro del caddie master. Mostra tutte le file del deposito che devono essere aperte, ordinate per priorita'.

Ordinamento per priorita'

Le file sono ordinate in base all'orario della prima partenza: la fila con il giocatore che parte prima viene mostrata per prima.

Colori di priorita'

Colore Significato
Rosso - Urgente, prime file da preparare
Giallo - Priorita' media
Verde - Puo' aspettare

Come usarla

  • Ogni card mostra: numero fila, conteggio sacche, orario prima partenza
  • Tap sulla card: espande e mostra i giocatori di quella fila con i relativi orari
  • Tap sul cerchio a destra: segna la fila come "preparata" (check verde). Le file preparate vanno in fondo alla lista
  • In alto: riepilogo con file totali, sacche totali e progresso preparazione

Vantaggi

Quando il caddie master apre una fila del deposito, prepara tutte le sacche di quella fila in una volta sola, evitando di tornare piu' volte nella stessa fila. L'app gli dice esattamente quante sacche ci sono e per quali giocatori.

Nota: i giocatori senza fila assegnata non compaiono in questa vista.

4. Anagrafica Giocatori

Gestisce l'elenco dei giocatori con il rispettivo numero di fila nel deposito.

Tre sotto-sezioni

Lista Aggiungi Importa

Lista

  • I giocatori sono raggruppati per numero di fila (fila 1, fila 2, ...)
  • Barra di ricerca per trovare un giocatore per nome
  • Tap sul numero fila di un giocatore: apre un popup per modificarla
  • X accanto al nome: elimina il singolo giocatore
  • In fondo: bottone Elimina tutta l'anagrafica (con doppia conferma)

Aggiungi

Inserisci manualmente un giocatore: nome completo (COGNOME NOME) e numero fila.

Importa

Importa molti giocatori in una volta da testo. Formato: una riga per giocatore.

ROSSI MARIO;5
BIANCHI LUIGI;12
VERDI ANNA;3

Separatori accettati: punto e virgola (;), virgola (,) oppure tab.

Gestione dati

Tutti i dati (anagrafica giocatori, gare, configurazione) sono salvati localmente sul dispositivo tramite localStorage.

  • Non serve connessione internet
  • Non serve un server
  • I dati persistono tra una sessione e l'altra
  • I dati sono legati al dispositivo: se si cancella la cache del browser o si disinstalla l'app, i dati vengono persi

Installazione

Android

  1. Trasferisci il file CaddieMaster.apk sul telefono (via email, Telegram, USB, ecc.)
  2. Apri il file APK sul telefono
  3. Se richiesto, abilita "Installa da fonti sconosciute" nelle impostazioni
  4. Completa l'installazione

iPhone

Per iPhone serve compilare il progetto con Xcode su un Mac con:

  1. Xcode installato (dall'App Store)
  2. Un Apple Developer Account (anche gratuito)
  3. iPhone collegato via USB
cd frontend
npm run build
npx cap sync
npx cap open ios

Da Xcode: selezionare il dispositivo e premere Play.

Aggiornamenti

Per aggiornare l'app dopo una modifica al codice:

cd frontend
npm run build          # ricompila il frontend
npx cap sync           # sincronizza con i progetti nativi

# Per Android:
export JAVA_HOME=/opt/homebrew/opt/openjdk@25/libexec/openjdk.jdk/Contents/Home
export ANDROID_HOME=~/Library/Android/sdk
cd android && ./gradlew assembleDebug

# L'APK aggiornato si trova in:
# android/app/build/outputs/apk/debug/app-debug.apk
