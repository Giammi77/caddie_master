Caddie Master

Gestione attrezzature deposito golf

Cos'e' Caddie Master

Caddie Master e' un'app per gestire la preparazione delle attrezzature da golf (sacche su carrelli) prima di una gara. Il deposito e' organizzato per file numerate, e ogni giocatore ha la propria attrezzatura in una fila specifica.

Data una lista di partenze, l'app dice al caddie master quali file aprire e quali attrezzature preparare, ottimizzando i viaggi al deposito.

Funziona offline Dati sul dispositivo Nessun server necessario

Installazione

Da browser (Android e iPhone)

Apri questo indirizzo dal browser del telefono:

https://giammi77.github.io/caddie_master

Poi aggiungi l'app alla schermata Home:

  • Android (Chrome): menu ⋮ → "Aggiungi a schermata Home"
  • iPhone (Safari): tasto condividi ⬆ → "Aggiungi a Home"

L'app apparira' con la sua icona e funzionera' anche senza connessione internet dopo il primo caricamento.

Android (APK nativo)

In alternativa, e' disponibile il file CaddieMaster.apk da installare direttamente:

  1. Trasferisci il file APK sul telefono (via email, Telegram, USB, ecc.)
  2. Apri il file APK sul telefono
  3. Se richiesto, abilita "Installa da fonti sconosciute" nelle impostazioni
  4. Completa l'installazione

Le 4 sezioni dell'app

La barra di navigazione in basso ha 4 tab:

+ Importa ⏰ Gara ☰ File 👤 Anagrafica

1. Importa Gara

Qui si importa la lista delle partenze di una gara.

Come funziona

1
Inserisci i dati
Compila il nome della gara, la data, e incolla il testo della lista partenze nella textarea.
2
Premi "Analizza"
L'app legge il testo e confronta i nomi con l'anagrafica. Ogni giocatore viene segnato come: Noto (fila gia' assegnata) oppure Nuovo (non presente in anagrafica).
3
Assegna le file ai nuovi (opzionale)
Per i giocatori nuovi puoi inserire il numero della fila. Se non lo conosci, puoi lasciarlo vuoto: il giocatore verra' semplicemente omesso dalla preparazione attrezzature nella vista File.
4
Premi "Conferma e Salva"
La gara viene salvata. I giocatori nuovi a cui hai assegnato una fila vengono aggiunti automaticamente all'anagrafica per le gare future.

Formato lista partenze

Il testo deve avere questo formato (campi separati da TAB):

08:36  1  COGNOME NOME 15  COGNOME NOME 17
08:45  1  COGNOME NOME 12  COGNOME NOME 8  COGNOME NOME 17

Ogni riga: ORARIO TEE GIOCATORE1 GIOCATORE2 ...

Il numero dopo il nome e' l'handicap e viene ignorato. Le voci "Privacy" vengono saltate automaticamente.

Gare precedenti

Le gare gia' importate appaiono in cima alla pagina. Puoi selezionarne una per rivederla oppure eliminarla con il bottone Elimina.

2. Gara

Mostra le partenze della gara selezionata, raggruppate per orario.

Funzionalita'

  • Ogni gruppo mostra orario e numero del tee di partenza
  • Accanto a ogni giocatore c'e' il N. fila
  • I giocatori senza fila mostrano No fila
  • Tap su un giocatore: evidenzia in verde tutti i giocatori della stessa fila in tutte le partenze
  • Tap di nuovo per deselezionare

Questa vista e' utile per capire rapidamente "chi parte con chi" e in quale fila si trovano le attrezzature.

3. File Deposito

Questa e' la vista principale per il lavoro del caddie master. Mostra tutte le file del deposito da aprire, ordinate per priorita'.

Ordinamento per priorita'

Le file sono ordinate in base all'orario della prima partenza: la fila con il giocatore che parte prima viene mostrata per prima.

Colori di priorita'

Colore Significato
Rosso — Urgente, prime file da preparare
Giallo — Priorita' media
Verde — Puo' aspettare

Come usarla

  • Ogni card mostra: numero fila, conteggio sacche, orario prima partenza
  • Tap sulla card: espande e mostra i giocatori di quella fila con i relativi orari di partenza
  • Tap sul cerchio a destra: segna la fila come "preparata" ✓. Le file preparate vanno in fondo alla lista
  • In alto: riepilogo con file totali, sacche totali e avanzamento della preparazione

Il vantaggio

Quando il caddie master apre una fila del deposito, prepara tutte le sacche di quella fila in una volta sola, evitando di tornare piu' volte nella stessa fila. L'app mostra esattamente quante sacche ci sono e per quali giocatori.

Nota: i giocatori senza fila assegnata non compaiono in questa vista.

4. Anagrafica Giocatori

Gestisce l'elenco dei giocatori con il rispettivo numero di fila nel deposito.

Tre sotto-sezioni

Lista Aggiungi Importa

Lista

  • I giocatori sono raggruppati per numero di fila (fila 1, fila 2, ...)
  • Barra di ricerca in alto per trovare un giocatore per nome
  • Tap sul numero fila di un giocatore: apre un popup per modificarla
  • X accanto al nome: elimina il singolo giocatore
  • In fondo alla lista: bottone Elimina tutta l'anagrafica (con doppia conferma di sicurezza)

Aggiungi

Inserisci manualmente un singolo giocatore: nome completo (COGNOME NOME) e numero fila.

Importa

Importa molti giocatori in una volta incollando un testo. Formato: una riga per giocatore.

ROSSI MARIO;5
BIANCHI LUIGI;12
VERDI ANNA;3

Separatori accettati: punto e virgola (;), virgola (,) oppure tab.

Gestione dati

Tutti i dati (anagrafica giocatori e gare) sono salvati localmente sul dispositivo.

  • Non serve connessione internet per usare l'app
  • Non serve un server
  • I dati rimangono salvati tra una sessione e l'altra
  • I dati sono legati al dispositivo: cancellando la cache del browser o disinstallando l'app, i dati vengono persi

Consiglio

Prima della prima gara, importa l'anagrafica completa dei giocatori con i numeri di fila. In questo modo, quando importi una lista partenze, la maggior parte dei giocatori verra' riconosciuta automaticamente e non dovrai assegnare le file manualmente.

Note tecniche

L'app e' realizzata con React e funziona interamente nel browser (nessun backend). I dati sono salvati in localStorage. Il progetto include anche la configurazione Capacitor per generare un APK Android nativo.

Aggiornamento dell'app

Per pubblicare una nuova versione:

cd frontend
npm run deploy     # compila e pubblica su GitHub Pages

Generare APK Android

cd frontend
npm run build
npx cap sync
export JAVA_HOME=/opt/homebrew/opt/openjdk@25/libexec/openjdk.jdk/Contents/Home
export ANDROID_HOME=~/Library/Android/sdk
cd android && ./gradlew assembleDebug

# APK in: android/app/build/outputs/apk/debug/app-debug.apk
