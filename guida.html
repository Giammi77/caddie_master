Caddie Master

Guida utente

A cosa serve

Caddie Master ti aiuta a preparare le attrezzature da golf prima di una gara.

Il deposito e' organizzato per file numerate. Ogni giocatore ha la propria sacca in una fila specifica. Quando ricevi la lista delle partenze di una gara, l'app ti dice:

  • Quali file aprire e in che ordine (le piu' urgenti prima)
  • Quante sacche ci sono in ogni fila
  • Per quali giocatori e a che ora partono

In questo modo prepari tutte le sacche di una fila in una volta sola, senza dover tornare piu' volte nella stessa fila.

Primo utilizzo

La prima volta che usi l'app, segui questi passaggi:

1
Importa l'anagrafica giocatori
Vai nella sezione Anagrafica → tab Importa.
Puoi caricare un file .txt con il pulsante Carica file .txt oppure incollare il testo direttamente. Formato: 
ROSSI MARIO;5
BIANCHI LUIGI;12
VERDI ANNA;3
2
Verifica l'anagrafica
Nel tab Lista controlla che i giocatori siano raggruppati correttamente per fila. Puoi correggere una fila con un tap sul numero verde.
3
Sei pronto!
D'ora in poi, quando importi una gara, l'app riconoscera' automaticamente i giocatori e le loro file.
Consiglio: Importa l'anagrafica completa prima della prima gara. Piu' giocatori sono in anagrafica, meno lavoro manuale avrai ad ogni gara.

Le 4 sezioni dell'app

In basso trovi la barra di navigazione con 4 tab:

+ Importa ⏰ Gara ☰ File 👤 Anagrafica

Importa Gara

Qui importi la lista delle partenze prima di ogni gara.

Passo per passo

1
Inserisci nome e data della gara
2
Carica o incolla la lista partenze
Usa il pulsante Carica file .txt per selezionare il file esportato dal sito web della federazione.
Le righe non conformi (intestazioni, righe vuote) vengono eliminate automaticamente.
Se il file contiene il nome della gara e la data nell'intestazione, i campi vengono compilati in automatico.

In alternativa puoi incollare il testo direttamente nella textarea.
3
Premi "Analizza"
L'app legge il testo e per ogni giocatore mostra:
  • Noto → lo conosce gia', fila assegnata automaticamente
  • Nuovo → non e' in anagrafica
4
Per i giocatori nuovi
Se conosci la fila, inseriscila nel campo a destra. Se non la conosci, lascia vuoto: il giocatore non apparira' nella preparazione attrezzature.
5
Premi "Conferma e Salva"
La gara e' pronta! I giocatori nuovi a cui hai assegnato una fila vengono salvati in anagrafica per le prossime gare.

Gare precedenti

In cima alla pagina trovi le gare gia' importate. Puoi:

  • Tap su una gara per riaprirla
  • Bottone Elimina per cancellarla

Gara

Mostra le partenze della gara, raggruppate per orario.

  • Ogni gruppo mostra l'orario e il tee di partenza
  • Accanto a ogni giocatore c'e' il numero fila
  • I giocatori senza fila mostrano No fila
Tap su un giocatore per evidenziare in verde tutti i giocatori che hanno la stessa fila nel deposito. Utile per vedere subito chi condivide la stessa fila. Tap di nuovo per deselezionare.

File Deposito

Questa e' la schermata principale per il tuo lavoro. Ti dice quali file aprire e in che ordine.

Come e' ordinata

Le file sono ordinate per urgenza: la fila con il giocatore che parte prima appare in cima alla lista.

Cosa significano i colori

 Rosso — Da preparare subito
Giallo — Priorita' media
Verde — C'e' ancora tempo

Come usarla

  • Ogni riga mostra il numero della fila, quante sacche contiene e l'orario della prima partenza
  • Tap sulla riga: si espande e mostra i nomi dei giocatori con i loro orari
  • Tap sul cerchio a destra ○ : segna la fila come preparata ✓. Le file completate vanno in fondo alla lista

Riepilogo in alto

In cima alla schermata trovi i contatori:

  • File — quante file devi aprire in totale
  • Sacche — quante sacche devi preparare in totale
  • Pronte — quante file hai gia' completato
  • Sacche OK — quante sacche sono pronte
Il vantaggio: Quando apri una fila del deposito, prepara tutte le sacche di quella fila in una volta sola. L'app ti dice esattamente quante sono e per chi. Cosi' non devi tornare piu' volte nella stessa fila.

Anagrafica Giocatori

Qui gestisci l'elenco dei giocatori con il loro numero di fila.

Tre tab

Lista Aggiungi Importa

Lista

I giocatori sono raggruppati per fila (fila 1, fila 2, ...).

  • Cerca: usa la barra in alto per trovare un giocatore per nome
  • Modifica fila: tap sul numero verde accanto al nome
  • Elimina: tap sulla X accanto al nome
  • Elimina tutto: bottone rosso in fondo (chiede conferma due volte)

Aggiungi

Per aggiungere un singolo giocatore: scrivi il nome (COGNOME NOME) e il numero della fila.

Importa

Per importare molti giocatori insieme. Usa il pulsante Carica file .txt per selezionare un file di testo, oppure incolla direttamente il testo. Formato: una riga per giocatore:

ROSSI MARIO;5
BIANCHI LUIGI;12
VERDI ANNA;3

Puoi usare come separatore: punto e virgola (;), virgola (,) oppure tab.

Informazioni utili

I dati restano sul telefono. L'anagrafica e le gare vengono salvate direttamente sul tuo dispositivo. Non serve connessione internet per usare l'app.
Attenzione: Se cancelli i dati del browser o disinstalli l'app, perdi l'anagrafica e le gare salvate. Tieni una copia dell'anagrafica nel file di testo originale come backup.
